## Multi-market expansion

The four backtest programs (sotorasib, adagrasib, vepdegestrant, BI-1701963) are each represented by their own LSLMSR contract instance. Addresses are populated in `frontend/lib/markets.ts`.

## Reproducing the deployment

```bash
cd contracts
forge install
forge test -vv --match-path test/Layer2.t.sol
source .env
forge script script/DeployLayer2.s.sol:DeployLayer2 --rpc-url base-sepolia --broadcast --verify
```
