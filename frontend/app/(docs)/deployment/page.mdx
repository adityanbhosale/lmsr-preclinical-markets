## Reproducibility

```bash
git clone https://github.com/adityanbhosale/lmsr-preclinical-markets
cd lmsr-preclinical-markets/contracts
forge install
forge test                          # 27/27 passing on Layer 1
forge script script/DeployLayer1.s.sol:DeployLayer1 --rpc-url sepolia
```

Frontend:

```bash
cd ../frontend
npm install
npm run dev                         # localhost:3000
```

CCTP demo (full Sepolia → Base → trade → claim → return round-trip) is documented in [`docs/cctp-demo.md`](https://github.com/adityanbhosale/lmsr-preclinical-markets/blob/main/docs/cctp-demo.md).
