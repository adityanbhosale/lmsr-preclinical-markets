# Dual-Layer Liquidity Infrastructure for Tokenized Biotech Milestones

A working architecture for **event-triggered** real-world assets — a distinct category from the yield-bearing RWAs that BUIDL, Securitize, Ondo, and JPMD have established as the foundation. Event-triggered RWAs (binary FDA outcomes, contingent milestone payments) need continuous price discovery before resolution, which the yield-bearing model wasn't designed to provide. The dual-layer architecture extends the lineage rather than replacing it.

## The architectural claim

Single-layer tokenization works for yield-bearing RWAs because the asset itself produces the cash flow being tokenized. Event-triggered RWAs break this pattern — the cash flow exists only at resolution, and prior to resolution the only meaningful market activity is **probability discovery**, not yield distribution. A single-layer design cannot host both a regulated security and a continuous prediction market on its outcome under the same compliance perimeter.

The dual-layer split:

- **Layer 1 (Ethereum mainnet / Sepolia testnet):** ERC-3643 SPV holding the legal claim. Permissioned, identity-gated, transfer-restricted. Behaves like Securitize-class infrastructure.
- **Layer 2 (Base / Base Sepolia):** LS-LMSR event contract referencing the public outcome. Permissionless, USDC-settled, continuously priced.
- **Bridge:** Circle CCTP — native USDC across the layer boundary, no wrapped bridge, no new institutional counterparty risk.

## Live testnet

Four markets currently live on Base Sepolia. Round-trip CCTP (Sepolia → Base → trade → claim → return) operationally proven.
