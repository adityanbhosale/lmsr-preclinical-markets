## Reproducing the deployment

```bash
git clone https://github.com/adityanbhosale/lmsr-preclinical-markets
cd lmsr-preclinical-markets/contracts
forge install
forge test -vv --match-path test/Layer1.t.sol      # 27/27 passing
source .env
forge script script/DeployLayer1.s.sol:DeployLayer1 --rpc-url sepolia
```

The dry-run uses ~6.75M gas total (≈0.00004 ETH at typical Sepolia prices). Broadcasting requires `--broadcast --verify -vvvv` and ~0.05 ETH headroom on the deployer wallet.
